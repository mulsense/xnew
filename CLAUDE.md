# xnew

Component-oriented programming library for JavaScript/TypeScript, designed for dynamic scenes and game development.

## Project Structure

- `src/` - Core library source code
- `examples/` - Example implementations
  - `1_basic/` - Basic usage examples
  - `2_pixi/` - PixiJS integration examples
  - `3_three/` - Three.js integration examples
  - `4_game/` - Game examples
- `dist/` - Built library files

## Key Features

- Component-oriented architecture
- Addons: xmatter, xpixi, xrapier2d, xthree

## Development

```bash
npm run build        # Build the library
npm run dev          # Watch mode
npm run build:addons # Build addons
npm test             # Run tests
```

## Tech Stack

TypeScript, Rollup, Jest
